# VMC850 立式加工中心机械维修与保养手册
**文档归属：** OEM 整机厂技术部
**适用机型：** VMC850 / VMC1060 系列
**警告：** 本手册仅针对纯机械与液压气动结构的排障。操作前必须切断机床动力电源！

---

## 故障代码：MECH-001 刀库换刀卡死（刀臂停在半空）
**现象说明：** 换刀过程中，ATC（自动换刀装置）的机械刀臂在拔刀或装刀时突然停止运动，停留在主轴和刀库之间的半空中。此时 CNC 系统可能无电气报警，按任何按键均无反应。
**可能原因：** 1. 车间气源气压突然下降，拔刀缸推力不足。
2. 刀柄锥孔内有铁屑或脏污，导致机械卡涩。
3. 刀臂电机刹车抱死或凸轮机构卡死。
**物理排除方法：**
1. 【安全第一】立即按下急停按钮，并切断机床主气源（关闭设备后方的气动三联件阀门）。
2. 使用 14 寸活口扳手，爬上机床顶部，找到刀库换刀电机的顶部六角螺母。
3. 人工用力旋转该六角螺母（俗称手动盘车），顺着电机外壳标注的箭头方向旋转，直到机械刀臂缓慢退回原位（刀库侧）。
4. 恢复气源，在面板上执行“刀库复位”指令。用无纺布清理主轴锥孔。

## 故障代码：MECH-002 主轴异响与严重发热
**现象说明：** 主轴在高速（>5000 rpm）旋转时发出尖锐的金属摩擦声，且主轴外壳温度摸起来明显烫手（>70℃）。此时 CNC 系统极有可能会报出“伺服驱动器负载过大”的相关代码。
**可能原因：**
1. 主轴内部前段陶瓷轴承严重磨损或保持架碎裂。
2. 主轴油冷机发生故障，冷冻液停止循环，导致热量无法散出。
**物理排除方法：**
1. 检查机床后方的主轴油冷机，观察冷媒压力表是否在正常的绿色区间，确认冷冻油管无折弯或泄漏。
2. 如果冷却系统正常但依然异响，极大概率为主轴轴承物理损坏。
3. 【操作规范】此故障无法通过修改软件参数解决！请准备内六角扳手拆卸主轴头外壳，用千分表测量主轴跳动（正常需小于 0.005mm）。若偏摆过大，需联系机床厂订购全新主轴总成并进行机械更换。

## 故障代码：MECH-003 排屑器履带卡死不转
**现象说明：** 机床加工产生的大量铁屑堆积在底盘。按下面板上的“排屑正转”按钮，排屑器不动作，并伴随电机沉闷的“嗡嗡”声或焦糊味。
**可能原因：**
1. 粗大的长条形铁屑、掉落的工件或废弃刀片，死死卡在了排屑器履带的链条齿轮中。
2. 排屑电机过载保护器（热继电器）受力跳闸。
**物理排除方法：**
1. 打开机床强电箱，检查并按压排屑器接触器下方的热继电器复位按钮（通常为蓝色或红色小点）。
2. 如果复位后按启动依然卡死，请立刻切断电源。
3. 戴上工业防割手套，使用铁钩或长嘴钳，在排屑器尾部出料口处，人工将卡住的铁屑块或异物强行拉出。
4. 重新上电，先点按“排屑反转”运行 3 秒松动间隙，再进行正转测试。

## 故障代码：MECH-004 导轨爬行与摩擦异响
**现象说明：** 机床 X/Y/Z 轴在低速进给切削时，出现“一顿一顿”的爬行共振现象，或者轴移动时发出干涩的金属摩擦声。加工出来的零件表面出现规律性的震纹。
**可能原因：**
1. 自动打油泵的润滑油箱缺油。
2. 内部润滑铜管被切屑砸扁，或分配器接头脱落，导致润滑油根本没有打入导轨滑块内部。
**物理排除方法：**
1. 检查机床侧面的润滑油箱液位，若低于下限，请补充 ISO VG68 导轨专用润滑油。
2. 若油箱有油，需拆开异响轴（如 X 轴）的伸缩钣金护罩（需使用 4mm 内六角扳手拆卸四周螺丝）。
3. 按下打油泵上的“手动强制打油”按钮，同时用手电筒观察导轨滑块上的油路接头是否有新鲜油滴渗出。如果发现断管，需截断并更换 4mm 铝塑油管，并锁紧黄铜卡套。
