# SINUMERIK 840D sl 排障手册（仿真数据）

适用版本：CNC 软件 V4.94 SP3 / SINAMICS S120 V5.2 SP2
文档编号：6FC5398-6BP40-6RA5

## NC 报警

### 2000 [通道 %1: ] 通道未准备就绪

参数：
- %1 = 通道号

说明：该通道未处于准备就绪状态，可能由程序错误、急停信号或硬件故障引起。

反应：NC 没有准备就绪。通道没有准备就绪。本通道 NC 启动禁止。已设置接口信号。报警显示。

排除方法：检查急停回路是否正常，确认 PLC 接口信号 DB10 DBX56.0 状态。如无急停信号，检查 NCU 模块 LED 指示灯是否正常。必要时重新上电。

程序继续：用复位键在该方式组中所有通道中删除报警。重新启动零件程序。

---

### 20010 [通道 %1: ] 程序段 %2 轴 %3 指令值溢出

参数：
- %1 = 通道号
- %2 = 程序段号
- %3 = 轴名称

说明：NC 程序中某轴的指令位置值超出系统允许的最大行程范围。常见于 G00 快速定位时坐标值写入错误，或工件坐标系偏移量设置过大。

反应：NC 停止。已设置接口信号。报警显示。报警时 NC 停止。

排除方法：
1. 检查零件程序中该程序段的坐标值是否正确。
2. 检查工件零点偏移（G54~G59）是否合理。
3. 检查机床数据 MD36100 $MA_POS_LIMIT_MINUS 和 MD36120 $MA_POS_LIMIT_PLUS 设置的软限位值。
4. 如使用了可编程零点偏移（TRANS/ATRANS），确认偏移量。

程序继续：用清除键或 NC-启动键清除报警。

### 20020 [轴 %1: ] 静态监控生效

参数：
- %1 = 轴名称

说明：轴在运动过程中实际位置偏离指令位置超过 MD36400 $MA_CONTOUR_TOL 设定的轮廓监控公差带。常见原因包括：
- 机械负载过大或卡死
- 伺服驱动参数调整不当
- 编码器反馈信号异常
- 联轴器松动或断裂

反应：本通道 NC 启动禁止。已设置接口信号。报警显示。报警时 NC 停止。触发 RE（快速回退）。

排除方法：
1. 检查机械传动系统是否正常，手动盘车确认无卡阻。
2. 检查伺服电机电流是否异常偏高（通过 SINAMICS 调试软件 Starter/SCOUT 查看 r0027）。
3. 检查编码器电缆连接是否牢固，屏蔽层是否可靠接地。
4. 调整速度环增益 MD32200 $MA_POSCTRL_GAIN 和加速度前馈 MD32800 $MA_VEFF_FFW_WEIGHT。
5. 适当增大轮廓公差 MD36400 $MA_CONTOUR_TOL（注意：仅用于诊断，不宜长期使用大公差）。

程序继续：用复位键在该方式组中所有通道中删除报警。重新启动零件程序。

### 20030 [轴 %1: ] 跟随误差过大

参数：
- %1 = 轴名称

说明：轴在插补运动中，实际位置与指令位置的差值超过 MD36500 $MA_ENC_COMP_TOL 设定的跟随误差极限。该报警通常在高速切削或大进给量时出现。可能原因：
- 伺服增益过低，动态响应不足
- 驱动器电流限制触发
- 机械间隙过大（反向间隙补偿 MD32450 $MA_BACKLASH 设置不当）
- 编码器信号干扰

反应：NC 停止。已设置接口信号。报警显示。

排除方法：
1. 检查伺服速度环增益 MD32200 $MA_POSCTRL_GAIN，适当提高（推荐范围 1~5）。
2. 检查加速度前馈 MD32800 $MA_VEFF_FFW_WEIGHT 是否设为 100%。
3. 使用圆度测试（圆插补）检查实际跟随误差曲线，调整增益。
4. 检查驱动器电流限值参数（p0640）是否过小。
5. 检查机械反向间隙并修正 MD32450 $MA_BACKLASH。

程序继续：用复位键在该方式组中所有通道中删除报警。重新启动零件程序。

### 25000 轴 %1 DRIVE-CLiQ：编码器故障

参数：
- %1 = 轴名称

说明：DRIVE-CLiQ 总线上检测到编码器通讯故障。可能原因：
- 编码器电缆断裂或接触不良
- DRIVE-CLiQ 集线器模块故障
- 编码器硬件损坏
- 总线终端电阻配置错误

反应：相关轴无法上使能。NC 停止。已设置接口信号。报警显示。

排除方法：
1. 检查编码器 DRIVE-CLiQ 电缆接头是否插紧，针脚有无弯曲或氧化。
2. 更换编码器电缆测试。
3. 检查 DRIVE-CLiQ 集线器模块（如 CX32-2）指示灯状态，绿色常亮为正常。
4. 使用 Starter/SCOUT 在线查看驱动对象的 r0029（通讯状态）。
5. 如更换电缆后仍报故障，更换编码器或伺服电机。
6. 检查总线拓扑配置（p0109~p0113）是否与实际硬件一致。

程序继续：故障排除并应答后，关闭/打开系统。

## SINAMICS 报警

### 207016 驱动：功率单元过温

说明：功率模块（电源模块或电机模块）内部温度超过阈值。温度传感器检测到散热器或 IGBT 模块温度过高。

信息值：组件号：%1，温度值：%2

驱动对象：所有驱动对象

反应：触发 OFF3（快速停车）。故障被记录在故障缓冲器中。

应答：消除故障原因后应答故障。

原因：
1. 环境温度过高（超过 40°C）
2. 风扇故障或风道堵塞
3. 散热器表面积灰严重
4. 功率模块过载运行（持续电流超额定值）
5. 制动电阻过热（再生能量过大）

排除方法：
1. 检查电柜内部温度，确认空调或风扇工作正常。
2. 清洁功率模块散热器表面积灰（断电后使用压缩空气）。
3. 检查功率模块内置风扇是否转动，必要时更换风扇。
4. 检查负载是否在额定范围内，查看 r0027（实际电流）与额定电流对比。
5. 检查制动电阻阻值及散热条件，确认制动占空比未超标。
6. 降低加工负载或增加间歇冷却时间。

应答方式：通过操作面板或 PLC 接口应答故障。

### 207020 驱动：直流母线过电压

说明：直流母线电压超过设定阈值。SINUMERIK 840D sl 阈值通常为 800V（单轴模块）或 750V（多轴模块）。

信息值：故障值：%1（实际直流母线电压，单位 0.1V）

驱动对象：所有驱动对象

反应：触发 OFF2（自由停车）。故障记录在故障缓冲器中。

应答：消除故障原因后应答故障。

原因：
1. 再生制动能量无法有效释放（制动电阻断路或制动斩波器故障）
2. 电源电压波动或突然升高
3. 减速时间过短，再生能量过大
4. 制动电阻参数配置错误（p1237 未激活）

排除方法：
1. 检查制动电阻接线是否牢固，用万用表测量阻值是否在标称范围内。
2. 检查制动斩波器使能参数 p1237 是否设置为 1。
3. 延长减速时间参数 p1121（斜坡下降时间）。
4. 检查进线电源电压是否在允许范围内（380V ±10%）。
5. 使用示波器监控直流母线电压波形（r0070），判断是否为瞬态过压。

应答方式：通过操作面板或 PLC 接口应答故障。

---

### 205005 驱动：I²t 功率单元过载

说明：功率单元的热模型计算出 I²t 值超过允许极限，表明功率模块已接近热保护限值。

信息值：故障值：%1（I²t 负载率，百分比）

驱动对象：所有驱动对象

反应：触发 OFF2（自由停车）。故障记录在故障缓冲器中。

应答：消除故障原因后应答故障。

原因：
1. 长时间过载运行
2. 切削参数过大（进给速度或切削深度超出机床能力）
3. 电机与功率模块匹配不当
4. 频繁启停导致热积累

排除方法：
1. 降低切削负载，减小进给速度或切削深度。
2. 检查电机铭牌数据与功率模块的匹配关系（p0307 电机额定功率 vs 功率模块额定电流）。
3. 增加工序间的冷却等待时间。
4. 查看 r0036（功率模块 I²t 负载率）监控当前热状态。
5. 如持续报警，考虑升级更大规格的功率模块。

应答方式：通过操作面板或 PLC 接口应答故障。

## PLC 报警

### 600001 [PLC %1: ] 用户报警：液压系统压力不足

参数：
- %1 = PLC 编号

说明：由 PLC 用户程序触发，表示液压系统压力低于设定阈值（通常由压力开关 B1 信号触发）。常见于：
- 液压泵电机未启动或过载跳闸
- 液压油箱油位过低
- 压力继电器设定值漂移
- 液压管路泄漏

反应：NC 停止。已设置接口信号。报警显示。

排除方法：
1. 检查液压泵电机是否运转，热继电器是否跳闸。
2. 检查液压油箱油位，必要时补充 46# 抗磨液压油。
3. 检查压力继电器设定值（通常设定为 4~6 MPa），必要时重新标定。
4. 检查液压管路及接头有无渗漏。
5. 检查溢流阀设定压力是否正确。

程序继续：故障排除后，通过 PLC 接口复位报警。

---

### 600010 [PLC %1: ] 用户报警：润滑系统故障

参数：
- %1 = PLC 编号

说明：集中润滑系统报警，通常由液位开关或压力开关触发。常见于：
- 润滑油箱油位过低
- 润滑泵堵塞或电机故障
- 润滑管路堵塞
- 分配器阀芯卡死

反应：NC 停止。已设置接口信号。报警显示。

排除方法：
1. 检查润滑油箱油位并补充。
2. 手动启动润滑泵，观察压力表是否建立压力。
3. 检查各润滑点出油情况，清理堵塞的分配器。
4. 检查润滑管路有无折弯或破裂。

程序继续：故障排除后，通过 PLC 接口复位报警。

## 循环报警

### 61000 [通道 %1: ] 循环 %2 参数 %3 值超出范围

参数：
- %1 = 通道号
- %2 = 循环名称（如 CYCLE81, CYCLE83 等）
- %3 = 参数名称

说明：调用固定循环时，输入的参数值超出允许范围。常见于钻孔循环的深度值、进给速度、安全高度等参数设置不当。

反应：程序段起始处的 NC 启动禁止。已设置接口信号。报警显示。

排除方法：
1. 检查循环调用语句中各参数的数值。
2. 参照编程手册确认参数允许范围。
3. 常见错误示例：
   - CYCLE83（深孔钻）中单次进给量 Q2 大于总深度 Q1
   - 安全平面 R2 高于参考平面 R3
   - 进给速度 F 为零或负值

程序继续：用清除键或 NC-启动键清除报警。

## HMI 报警

### 400000 通讯：NCK 连接中断

说明：HMI 操作面板与 NCU 之间的通讯中断。可能原因：
- MPI/PROFINET 通讯电缆松动或断裂
- NCU 模块死机或重启
- 通讯参数配置错误
- 网络交换机故障

反应：HMI 显示通讯错误，无法操作机床。

排除方法：
1. 检查 NCU 前面板 LED 指示灯状态。
2. 检查 MPI/PROFINET 通讯电缆连接。
3. 重启 NCU（断电等待 30 秒后重新上电）。
4. 检查 HMI 通讯参数设置（站号、波特率、IP 地址）。
5. 使用 SINUMERIK 诊断页面查看通讯状态。

程序继续：通讯恢复后自动消除。

## 系统错误

### 1000 系统错误 %1 %2 %3 %4

参数：
- %1 = 系统出错号
- %2 = 生成的参数
- %3 = 生成的参数
- %4 = 生成的参数

说明：该报警指出系统内部的错误状态，并包含错误编号，指出错误原因和错误地点。系统错误通常为软件内部异常，非用户可自行排除的故障。

反应：NC 没有准备就绪。通道没有准备就绪。本通道 NC 启动禁止。已设置接口信号。报警显示。报警时 NC 停止。

排除方法：出现该系统错误时，请联系西门子技术支持中心。www.siemens.com/sinumerik/help

为方便快速处理，请提供以下信息：
- 报警号以及报警文本
- 发出报警信息前的操作/运行方式说明
- 按下快捷键 Ctrl + Alt + D 创建日志文件

程序继续：关闭/打开系统。

## 排障流程速查表

| 故障现象 | 可能报警号 | 排查方向 |
|---------|-----------|---------|
| 轴无法上使能 | 25000, 207020 | 检查 DRIVE-CLiQ 通讯、编码器、驱动使能链 |
| 加工精度差 | 20020, 20030 | 检查伺服增益、反向间隙、编码器 |
| 运行中突然停机 | 207016, 207020, 205005 | 检查电源电压、制动电阻、散热 |
| 程序无法运行 | 2000, 61000, 4009 | 检查急停、程序参数、机床数据 |
| 系统无法启动 | 1000, 1005 | NCU 硬件故障或系统软件损坏，联系西门子 |
| 液压/润滑报警 | 600001, 600010 | 检查泵、油位、压力开关、管路 |
